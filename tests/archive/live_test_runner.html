Problem-Tests

Test 1: "Neue Unterhaltung" Button und Tabs

Prüft, ob der "Neue Unterhaltung" Button korrekt funktioniert und keine doppelten aktiven Tabs erzeugt.

Test 2: Session-Eingabefeld Test

Prüft, ob Texteingaben pro Tab/Session gespeichert und wiederhergestellt werden.

Test 3: Text-Stream Test

Prüft, ob der Text-Stream Mechanismus funktioniert und Antworten inkrementell angezeigt werden.

Test 4: Admin-Tab Test

Prüft, ob der Admin-Tab korrekt geladen wird beim ersten Klick.

Test 5: System-Tab Test

Prüft, ob Systemstatistiken im Admin-System-Tab korrekt angezeigt werden.

Test 6: Feedback-Tab Test

Prüft, ob Feedback-Statistiken im Admin-Feedback-Tab korrekt angezeigt werden.

Test 7: MOTD-Tab Test

Prüft, ob die MOTD-Vorschau im Admin-MOTD-Tab korrekt formatiert wird.

Test 8: Dokumentenkonverter-Tab Test

Prüft, ob der Dokumentenkonverter-Tab korrekte Konvertierungsoptionen anzeigt.

Test 9: Feedback-Buttons Test

Prüft, ob die Feedback-Buttons (Daumen hoch/runter) am unteren Rand der Nachrichten funktionieren.
